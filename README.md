## ✨ 插件功能

* 高度自定义 Gui 界面
* 支持 MySQL, YAML 格式存储
* 自定义商品原信息是否显示
* 自定义商品所在槽位
* 个人仓库及仓库面板自定义
* 支持版本: 1.8-1.20
* 支持多种货币(Vault, PlayerPoints, NyEconomy)
* 支持商品分类、商品搜索、商品黑名单
* 支持指定商品/分类价格范围
